# SmartGas App Web V1

Estructura del frontend, el layout principal, el router, i18n en inglés/español.

## Ejecutar el proyecto

```bash
npm install
npm run start
```

También se puede ejecutar por separado:

```bash
npm run server
npm run dev
```

Usuario de prueba para el `db.json`:

```txt
usuario@smartgas.com / smartgas
```

## Distribución sugerida

Persona 1: base del proyecto, layout, toolbar, router, dashboard e i18n.

Persona 2: iam.

Persona 3: kitchen-monitoring.

Persona 4: incident-detection e incident-prevention-notification.

Persona 5: post-incident-procedures y payment-management.
